This is a simple C program that calculates end time by adding a given amount of time to a start time.
    Valid program arguments:
        <value> : Additive input should be formatted as [HH.xx]. (Round hours to the nearest 100th)
        -s : Flag that you want to pull system time for the start time
        -p : Flag that you want to enter a specified rounding value
        -c : Return result in military time
        -h : Hides the manual for the program
        -d : Print detailed debugging info to stderr
